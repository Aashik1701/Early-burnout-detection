Risk Engine Pipeline

From raw probabilities to personalized, priority-ranked interventions.

1
Risk Score Normalization

Convert raw model probability (0.0-1.0) into a user-friendly 0-100 score.

Out: Score (e.g., 87)
2
Level Classification

Apply adaptive threshold bins based on operating mode (Balanced/High-Recall).

Out: High Risk
3
Trigger Extraction

Calculate SHAP values to identify top-5 features driving the risk score.

Out: [Attendance, GPA, Stress...]
4
Label Generation

Map technical features to human-readable "Disengagement Indicators".

Out: "Attendance Decline"
5
Intervention Assignment

Match dominant trigger to the intervention catalog using priority rules.

Final: Engagement Counseling
Intervention Priority Map

First-match wins logic ensures the most critical behavioral signal is addressed first.
